## 🚀 Setup Instructions

1. **Create GitHub repository:**
   ```bash
   git init
   git add .
   git commit -m "Initial commit: Add legal documents for Kash Cube"
   git branch -M main
   git remote add origin https://github.com/vloganathane/kashcube-privacy.git
   git push -u origin main
   ```

2. **Enable GitHub Pages:**
   - Go to your repo: `https://github.com/vloganathane/kashcube-privacy`
   - Click **Settings** → **Pages**
   - Under **Source**, select: **main** branch, **/ (root)** folder
   - Click **Save**
   - Wait 1-2 minutes for deployment

3. **Verify:**
   - Visit: `https://vloganathane.github.io/kashcube-privacy/`
   - Check all links work (privacy.html, terms.html)

4. **Add to Play Store:**
   - Copy the privacy URL: `https://vloganathane.github.io/kashcube-privacy/privacy.html`
   - Paste in Play Console → **Store settings** → **Privacy Policy URL**

## 📱 About Kash Cube

Kash Cube is a **privacy-first financial tracker** built for India:
- 100% local-only data (SQLite)
- No cloud sync, no servers
- UPI/bank SMS auto-capture
- GST invoice generation
- Credits/Udhar tracking
- Encrypted backups (AES-256)

**Platform:** Android (Google Play Store)  
**Version:** 1.0.0  
**License:** Closed source (proprietary)

## 🔒 Privacy Commitment

All financial data stays on your device. No network calls to transmit transaction data. Optional analytics (Firebase) is opt-in only and never includes financial information.
